## 🎯 Usage Guidelines

### Primary Color (#001645)
**Use for:**
- Main navigation
- Primary buttons
- Headings
- Important text
- Focus states

**Don't use for:**
- Large text blocks
- Backgrounds (except dark mode)

### Secondary Color (#17BC43)
**Use for:**
- Success messages
- Call-to-action buttons
- Highlights
- Positive feedback
- Hover states

**Don't use for:**
- Error messages
- Warning states

### Muted Color (#F5F7FA)
**Use for:**
- Section backgrounds
- Card backgrounds
- Disabled states
- Subtle separators

**Don't use for:**
- Primary content areas
- Important information

## 📱 Responsive Color Usage

### Mobile
- Prefer high contrast colors
- Use larger touch targets with primary colors
- Ensure text is readable on all backgrounds

### Tablet
- Balance between mobile and desktop
- Use color to guide user flow
- Maintain consistency with mobile

### Desktop
- Full color palette available
- Use subtle hover states
- Leverage color for visual hierarchy
